# youtube-clone
This is a clone of YouTube, developed as a modern web application using the technologies I have learned. It offers a more user-friendly interface and improved performance compared to the original.
 
# TECHNOLOGIES
- axios
- millify
- moment
- react
- react-dom
- react-icons
- react-player
- react-router-dom
